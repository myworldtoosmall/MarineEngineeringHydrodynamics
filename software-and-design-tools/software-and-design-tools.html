Виртуальный центр проектирования

Набор вспомогательных инструментов для предварительного проектирования судов.

Обеспечивает создание гидродинамически точных прототипов и моделей судов для морских симуляторов реального времени.

Симулятор динамики судна

Компактный полнофункциональный симулятор динамики судов и плавающих объектов.

Обеспечивает высокоточное и более быстрое, чем в реальном времени, численное моделирование маневренности и мореходности судна для различных условий окружающей среды.

Может использоваться с целью гидродинамической верификации проектных решений для прототипов судов, созданных с помощью Виртуального центра проектирования, и как "движок" для разработки морских симуляторов и игровых приложений.
